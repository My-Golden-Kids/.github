## 🧱 System Architecture
.
  
> 시니어 사용자의 음성 및 OCR(영수증 스캔) 입력을 기반으로 한 하이브리드 모바일 앱 시스템 구성도입니다.
> Next.js 기반의 프론트엔드와 iOS HealthKit API를 네이티브 컨테이너로 통합했으며,
> Spring Boot 백엔드는 Gemini AI(대화형 STS), Google Vision OCR, 마이데이터 연동 서비스를 처리합니다.
